### Code

```
fonction ValeurAbsolue(x: réel) : réel
    Variables: y:réel
    Début
        Si x < 0 alors
            y <- -x
        Sinon
            y <- x
        Fin Si
        Renvoyer y
    Fin

fonction MoitiéRésidu(x: réel) : réel
    Variables: y:réel
    Début
        y <- ValeurAbsolue(x/2)
        Tant que y > 1 faire
            y <- ValeurAbsolue(x/2)
        Fin Tant que
        Renvoyer y
    Fin

Début
    Saisir(x)
    y <- MoitiéRésidu(x)
    Afficher(y)
Fin
```
